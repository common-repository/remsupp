== Installation ==

RemSupp WordPress plugin installation:

1. Log into your WordPress admin dashboard.
2. Go to “Plugins” using the sidebar on the left.
3. Click “Add New” and find “RemSupp” with the search bar.
4. Install the plugin and activate it.
5. The RemSupp icon will appear on the left sidebar of your WP admin panel.

After activating the plugin, create your account and start the product tour.
